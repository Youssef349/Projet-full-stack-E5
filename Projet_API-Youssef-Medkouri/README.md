# Readme

## Setup
1. Build the image: `./build.sh`
2. Run the image in a container called 'restaurant': `./run.sh`
3. If running for the first time, create the database: `./setup.sh`

## Usage
* You can visit the admin panel at [http://0.0.0.0:8000/admin](http://0.0.0.0:8000/admin) and login with username: admin, password: admin
  * The admin panel will you help you create and experiment with objects 
* You can view the API routes here at: [http://0.0.0.0:8000/docs](http://0.0.0.0:8000/docs)
